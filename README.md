# World Wide - Travel Tracker | https://world-wise1.web.app/

World Wide is a React-based travel tracker application that allows users to record their travels by selecting locations on a map, specifying travel dates, and adding personal notes to each travel entry. The project uses Firebase for authentication and Firestore for data storage.

## Features

- **Map Interaction:** Easily select travel locations by interacting with the map.
- **Date Picker:** Choose travel dates conveniently using the date picker component.
- **Notes Section:** Add personal notes to each travel entry for memorable moments.
- **Firebase Integration:** Secure authentication and cloud storage with Firebase services.

## Technologies Used

- **React + Vite:** A JavaScript library for building user interfaces.
- **React-Router-Dom:** Declarative routing for React.js.
 - **Context API + useReducer**: Used to manage state and share data between components efficiently.
- **Firebase (Firestore, Authentication, Hosting):** A comprehensive suite of cloud-based services, including authentication and Firestore for data storage.
- **Leaflet:** An open-source JavaScript library for interactive maps.
- **React-Leaflet:** A React wrapper for Leaflet, providing a set of React components for common Leaflet map elements.
- **jwt-decode**: A library for decoding JSON Web Tokens (JWTs).
- **React-Datepicker**: A flexible datepicker component for React.
- **UUID**: A library for generating unique identifiers.

## Getting Started

### Prerequisites

- Node.js and npm: Make sure you have Node.js installed on your machine.

### Installation

Follow these steps to set up and run the World Wide project locally:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/NadaAly56/CityNote-worldWide.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd CityNote-worldWide
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```

4. **Run the development server:**

    ```bash
    npm run dev
    ```

5. **Open your browser and visit [http://localhost:3000](http://localhost:3000).**

Now you're all set to explore and use World Wide locally on your machine!

### Usage

World Wide is a travel tracker application where you can record your travels. Follow the steps below to use the application:

1. **Sign Up or Log In:**
   - Create an account or log in if you already have one.

2. **Explore the Map:**
   - Navigate the map to find the locations you've visited or plan to visit.

3. **Add a Travel Note:**
   - Click on a location on the map.
   - Enter the date and any additional notes about your travel.
   - Save the travel note.

4. **View Your Travels:**
   - See a list of all your recorded travels with details.

5. **Enjoy Your Travel Diary:**
   - Keep your travel memories organized and easily accessible.
